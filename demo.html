Features

Radar Chart

Visualize all 7 PPA channel scores in an intuitive spider chart.

Oscilloscope Waveform

Clock drift data rendered as a real-time style waveform.

Cache Latency Heatmap

4x4 color-coded heatmap showing cache timing profiles.

Jitter Constellation

Star map visualization of instruction jitter patterns.

Architecture-Aware

Visually distinct cards for different architectures (x86, ARM, PowerPC, etc.)

SVG Output

Scalable vector graphics output for crisp rendering at any size.

Quick Start

Installation

# Clone the repository git clone https://github.com/rmartinppa/ppa-attestation-visualizer.git cd ppa-attestation-visualizer # No dependencies required! Uses only Python standard library python3 ppa_visualizer.py sample-fingerprint.json output.svg

Python API

from ppa_visualizer import PPAVisualizer import json # Load fingerprint data with open('fingerprint.json') as f: data = json.load(f) # Generate visualization viz = PPAVisualizer(data) svg_output = viz.generate_svg() # Save to file with open('identity.svg', 'w') as f: f.write(svg_output)

Sample Input Format

{ "hardware_id": "g4_mac_mini_1.42ghz", "architecture": "PowerPC G4", "overall_score": 0.82, "channels": { "clock_drift": { "score": 0.87, "waveform": [...] }, "cache_timing": { "score": 0.92, "heatmap": [...] }, "simd_identity": { "score": 0.78 }, "thermal_drift": { "score": 0.65 }, "instruction_jitter": { "score": 0.71, "constellation": [...] }, "anti_emulation": { "score": 0.95 }, "fleet_detection": { "score": 0.88 } } }
